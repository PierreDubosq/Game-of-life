# Game-of-life
A simple game of life with the Raylib

## How to run

```bash
$ git clone https://github.com/PierreDubosq/Game-of-life
$ cd Game-of-life
$ make
$ ./game_of_life -h
```

## Libraries

- [Raylib 5.0](https://www.raylib.com/)
